# Modules

Jotain organizes custom functionality into modules located in the `lisp/` directory. Each module follows Emacs Lisp conventions with the `jotain-` prefix.

## Module Conventions

- Files are placed in `lisp/` and named `jotain-<feature>.el`
- Each file uses `lexical-binding: t`
- Each file provides its feature with `(provide 'jotain-<feature>)`
- Test files go in `test/` as `jotain-<feature>-test.el`

## Current Modules

### jotain-telemetry

Telemetry and metrics collection module. Enabled via a minor mode:

```elisp
(use-package jotain-telemetry
  :custom
  (jotain-telemetry-enabled t)
  :config
  (jotain-telemetry-mode 1))
```

## Adding New Modules

1. Create `lisp/jotain-<feature>.el` with the standard header
2. Add `(provide 'jotain-<feature>)` at the end
3. Load it from `init.el` using `use-package`
4. Create corresponding tests in `test/jotain-<feature>-test.el`
