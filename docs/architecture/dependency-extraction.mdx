## How it works

`nix/lib/dependencies.nix` uses pure Nix regex to scan all `.el` files in `elisp/` for `(use-package PACKAGE-NAME` declarations and maps each package name to its nixpkgs equivalent.

### Extraction pipeline

1. **Scan**: Find all `(use-package ...)` forms in `.el` files
2. **Filter**: Exclude packages with `:ensure nil`
3. **Map**: Convert Elisp package names to nixpkgs `emacsPackages` attribute names
4. **Resolve**: Look up actual packages from the `emacsPackages` set

### Available functions

| Function | Purpose |
|----------|---------|
| `extractUsePackages` | Extract package names from a single `.el` file |
| `scanDirectory` | Find all `.el` files and extract packages from all of them |
| `getPackagesForDirectory` | Get Emacs packages ready for `emacsWithPackages` |
| `mapToNixpkgsName` | Map an Elisp package name to its nixpkgs equivalent |
| `listPackageNames` | Debug utility to see extracted packages |

## Controlling extraction

### Auto-extracted (default)

Any `use-package` form is auto-extracted if the package name matches a nixpkgs attribute:

```elisp
(use-package magit
  :bind ("C-x g" . magit-status))
;; => Automatically installs emacsPackages.magit
```

### Exclude built-in packages

Use `:ensure nil` for packages built into Emacs:

```elisp
(use-package project
  :ensure nil
  :config ...)
;; => Not extracted, already part of Emacs
```

### Name mismatches

When the Elisp package name doesn't match the nixpkgs attribute, add a mapping in `packageNameMap`:

```nix
# In nix/lib/dependencies.nix
packageNameMap = {
  "some-elisp-name" = "actual-nixpkgs-name";
  "built-in-thing" = null;  # null = skip entirely
};
```

## Usage in Nix

```nix
let
  jotainLib = import ./nix/lib { inherit pkgs lib; };

  myEmacs = (pkgs.emacsPackagesFor pkgs.emacs).emacsWithPackages (epkgs:
    jotainLib.dependencies.getPackagesForDirectory ./elisp epkgs
  );
in myEmacs
```

## Debugging

List all extracted package names:

```bash
nix eval --impure --expr '
  let
    lib = import ./nix/lib {
      pkgs = import <nixpkgs> {};
      lib = (import <nixpkgs> {}).lib;
    };
  in lib.dependencies.listPackageNames ./elisp
'
```
