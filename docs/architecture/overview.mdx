---
title: Architecture Overview
description: How Jotain's components fit together
---

Jotain is structured as a Nix Flake that builds a fully configured Emacs distribution. The architecture has two main layers: Elisp modules (the Emacs configuration) and the Nix build system (packaging and dependency management).

## Directory structure

```
flake.nix                       # Flake entry point (flake-parts + emacs-overlay)
├── init.el                     # Emacs init — loads all modules in order
├── early-init.el               # Early init — startup optimization, tree-sitter paths
├── elisp/                      # Feature modules (one per domain)
├── tests/                      # ERT test suite
├── nmt-tests/                  # Nix Module Tests for Home Manager
├── emacs.nix                   # Emacs 30 (PGTK) build with packages + runtime deps
├── default.nix                 # Jotain package definition
├── shell.nix                   # Development shell
├── nix/
│   ├── lib/
│   │   ├── dependencies.nix    # Auto-extracts packages from use-package declarations
│   │   └── runtime-deps.nix    # LSP servers, fonts, CLI tools, tree-sitter grammars
│   ├── overlays/
│   │   └── default.nix         # Builds jotain-modules + custom packages
│   └── modules/
│       ├── home/default.nix    # Home Manager module
│       └── nixos/              # NixOS module
└── justfile                    # Development commands
```

## Data flow

<Steps>
  <Step title="Nix evaluates the flake">
    `flake.nix` uses `flake-parts` and `nix-community/emacs-overlay` for bleeding-edge Emacs builds.
  </Step>
  <Step title="Overlay builds elisp as a package">
    `nix/overlays/default.nix` compiles `elisp/` into an Emacs package called `jotain-modules` via `trivialBuild`.
  </Step>
  <Step title="Dependencies are auto-extracted">
    `nix/lib/dependencies.nix` scans all `.el` files for `use-package` declarations and maps them to nixpkgs Emacs packages.
  </Step>
  <Step title="Emacs is built with everything bundled">
    `emacs.nix` combines Emacs 30 (PGTK) + auto-extracted packages + `jotain-modules` + runtime dependencies.
  </Step>
  <Step title="Home Manager wires it up">
    `nix/modules/home/default.nix` installs the built Emacs, sets up config files, and optionally enables the daemon service.
  </Step>
</Steps>
