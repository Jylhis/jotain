Jotain uses `treefmt-nix` for consistent formatting across all file types.

## Running the formatter

```bash
just format
# or
nix fmt
```

## Formatters

| File type | Formatter |
|-----------|-----------|
| `.nix` | `nixpkgs-fmt` |
| Shell scripts | `shfmt` + `shellcheck` |
| `.el` | Emacs built-in `indent-region` |

## CI integration

Formatting is checked as part of `nix flake check`. The CI will fail if files are not properly formatted.

Run `just check-instant` (< 10 seconds) for a quick formatting + smoke test before pushing.
