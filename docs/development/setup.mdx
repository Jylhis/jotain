title: Development Setup
description: Set up a development environment for contributing to Jotain
---

## Enter the dev shell

All project tools are available in the Nix development shell:

```bash
nix develop
```

This provides: Emacs, LSP servers, formatters, `just`, and project-specific wrappers like `emacs-dev`.

The dev environment uses an isolated `.dev-home/` directory for Emacs configuration, so your personal setup is never affected.

## Running Emacs

```bash
# Run Emacs with project config in isolated .dev-home (recommended)
just emacs-dev

# Same, with console output for debugging
just emacs-test-interactive

# Run without isolation (use with caution)
just emacs-clean
```

## Try without installing

Test the built package in a temporary environment:

```bash
nix run
```

## Clean up

```bash
just clean  # Remove .elc files, caches, and .dev-home
```
