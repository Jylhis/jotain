The `ai` module (`elisp/ai.el`) integrates AI coding assistants into the Emacs workflow.

## Claude Code IDE

[claude-code-ide](https://github.com/manzaltu/claude-code-ide.el) provides Claude AI integration directly within Emacs. This is a custom package built via `trivialBuild` in the Nix overlay since it's not available in nixpkgs.
