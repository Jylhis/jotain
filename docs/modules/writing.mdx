title: Writing
description: Org-mode and documentation tools
---

The `writing` module (`elisp/writing.el`) configures Org-mode and other documentation tools.

## Org-mode

Emacs's built-in [Org-mode](https://orgmode.org/) is configured with sensible defaults for note-taking, task management, and document authoring.

## Markdown

Markdown editing support with `multimarkdown` as the rendering backend (provided via Nix runtime dependencies).

## Utility functions

The `utils.el` helper module is loaded by `writing.el` and provides shared utility functions used by the writing tools.
