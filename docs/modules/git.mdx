The `git` module (`elisp/git.el`) provides Git integration through Magit and supporting packages.

## Magit

[Magit](https://magit.vc/) is the primary Git interface, providing a complete porcelain for Git operations.

| Key | Action |
|-----|--------|
| `C-x g` | Open Magit status |

## diff-hl

`diff-hl` highlights uncommitted changes in the fringe (gutter) of file buffers, showing added, modified, and deleted lines at a glance.

## CLI tools

The following Git-related CLI tools are provided by the runtime dependencies:

- **git** — Required by Magit
- **ripgrep** — Fast search used by `consult-ripgrep` and `xref`
- **fd** — Fast find used by `consult-fd`
