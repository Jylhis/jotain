The `core` module (`elisp/core.el`) configures Emacs fundamentals — sensible defaults, keybindings, and built-in modes.

## Key settings

- Line and column numbers in the mode line
- Smooth scrolling
- Auto-save and backup file management via `no-littering`
- Clipboard integration with the system
- Electric pair mode for auto-closing brackets
- Winner mode for window layout undo/redo
- Recentf for recent file tracking
- Which-key for keybinding discovery

## Keybindings

| Key | Action |
|-----|--------|
| `C-x j` | Toggle window split direction |
| `C-c u` | Winner undo (restore previous window layout) |
| `C-c r` | Winner redo |

## Built-in modes enabled

- `electric-pair-mode` — Auto-close brackets and quotes
- `winner-mode` — Undo/redo window configurations
- `recentf-mode` — Track recently opened files
- `savehist-mode` — Persist minibuffer history
- `which-key-mode` — Display available keybindings
