# Packages

Jotain uses a minimal set of external packages, preferring built-in Emacs functionality where possible.

## External Packages

| Package | Source | Purpose |
|---|---|---|
| `nix-ts-mode` | MELPA | Tree-sitter based major mode for Nix files |
| `markdown-mode` | MELPA | Major mode for Markdown files, with GFM support |

## Tree-sitter Grammars

When built with `withTreeSitterGrammars` (the default), Jotain includes all 275 tree-sitter grammars from nixpkgs. These are automatically available via `treesit-extra-load-path` configured in the Nix-generated `site-start.el`.

## Adding Packages

To add a new package, add a `use-package` declaration in `init.el`:

```elisp
(use-package package-name
  :ensure t
  :config
  ;; configuration here
  )
```

For packages that should be associated with file types, use `:mode`:

```elisp
(use-package some-mode
  :ensure t
  :mode "\\.ext\\'")
```
