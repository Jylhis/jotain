---
title: Daemon Mode
description: Running Emacs as a background service
---

By default, Jotain runs Emacs as a systemd service (Linux) or launchd service (macOS). This keeps Emacs running in the background so client connections are instant.

## Managing the service

```bash
# Check status
systemctl --user status emacs.service

# Restart after config changes
systemctl --user restart emacs.service

# View logs
journalctl --user -u emacs.service
```

## Connecting to the daemon

```bash
# New GUI frame
emacsclient -c

# Terminal frame
emacsclient -t

# Open a file
emacsclient -c /path/to/file
```

## Disabling the daemon

If you prefer to launch Emacs directly:

```nix
programs.jotain = {
  enable = true;
  enableDaemon = false;
};
```

Then start Emacs normally with `emacs`.

<Note>
  When running as a daemon, themes are applied after the first frame is created via `server-after-make-frame-hook` to ensure correct rendering.
</Note>
