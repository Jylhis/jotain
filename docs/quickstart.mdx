## Prerequisites

- [Nix](https://nixos.org/download.html) with flakes enabled
- [Home Manager](https://github.com/nix-community/home-manager) (standalone or as a NixOS module)

## Installation

### 1. Add Jotain to your flake inputs

```nix
{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
    home-manager.url = "github:nix-community/home-manager";
    jotain.url = "github:Jylhis/jotain";
  };
}
```

### 2. Enable the Home Manager module

```nix
{
  home-manager.users.<your-username> = {
    imports = [ jotain.homeModules.default ];

    programs.jotain = {
      enable = true;
      # enableDaemon = true;      # default: runs as systemd service
      # includeRuntimeDeps = true; # default: installs LSP servers, fonts, CLI tools
    };
  };
}
```

<Accordion title="Standalone Home Manager (without NixOS)">
```nix
{
  outputs = { nixpkgs, home-manager, jotain, ... }: {
    homeConfigurations."<your-username>" = home-manager.lib.homeManagerConfiguration {
      pkgs = nixpkgs.legacyPackages.x86_64-linux;
      modules = [
        jotain.homeModules.default
        {
          programs.jotain.enable = true;
        }
      ];
    };
  };
}
```
</Accordion>

### 3. Rebuild

```bash
# NixOS
sudo nixos-rebuild switch

# Standalone Home Manager
home-manager switch --flake .
```

### 4. Launch

```bash
# If daemon is enabled (default):
emacsclient -c

# Or launch directly:
emacs
```

## Try without installing

You can try Jotain in an isolated temporary environment without affecting your system:

```bash
nix run github:Jylhis/jotain
```

## Verification checklist

After installation, verify everything is working:

<Steps>
  <Step title="Emacs starts cleanly">
    Check the `*Messages*` buffer for errors.
  </Step>
  <Step title="Dashboard appears">
    The Enlight dashboard should display on startup.
  </Step>
  <Step title="Tree-sitter works">
    Open a `.nix` file — syntax highlighting should use tree-sitter.
  </Step>
  <Step title="Completion works">
    Type a partial identifier — Corfu completion popup should appear.
  </Step>
  <Step title="Git works">
    Press `C-x g` in a Git repository — Magit status should open.
  </Step>
  <Step title="Theme toggle works">
    Press `C-c t` — theme should switch between light and dark.
  </Step>
</Steps>
