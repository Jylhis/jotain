# Quickstart

## Clone the Repository

```bash
git clone https://github.com/Jylhis/jotain.git
cd jotain
```

## Build and Run

```bash
# Build Emacs with Nix
just linux

# Run Emacs with the Jotain config
just run
```

## Development Environment

Jotain includes a [devenv](https://devenv.sh) configuration for development:

```bash
devenv shell
```

## Clean Build Artifacts

```bash
just clean
```

This removes `eln-cache/` and `result/` directories.
