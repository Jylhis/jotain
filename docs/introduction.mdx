Jotain is a modern, modular Emacs configuration managed with [Nix Flakes](https://nixos.wiki/wiki/Flakes), emphasizing performance and reproducible development. It provides a complete development environment with LSP support, modern completion, Git integration, and sensible defaults.

## What's included

Jotain bundles a curated set of packages and tools:

- **Completion**: Vertico, Corfu, Consult, Orderless
- **Programming**: Eglot (LSP), tree-sitter, DAP debugging
- **Git**: Magit, diff-hl
- **UI**: Nord theme system with light/dark toggle, Enlight dashboard
- **Writing**: Org-mode, Markdown support
- **AI**: Claude Code IDE integration
- **Platform support**: Linux, macOS, Android

## How it works

All Emacs packages are installed via Nix — never via MELPA or ELPA at runtime. The project exposes a `homeModules.default` for [Home Manager](https://github.com/nix-community/home-manager) integration, making installation a single line in your Nix configuration.
