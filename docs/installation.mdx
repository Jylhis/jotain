# Installation

Jotain uses [Nix](https://nixos.org/) to build Emacs from source. You need Nix installed on your system.

## Prerequisites

- [Nix package manager](https://nixos.org/download)
- [just](https://github.com/casey/just) command runner (optional, for convenience commands)

## Building Emacs

The simplest way to build is with `just`:

```bash
just linux
```

This runs `nix-build --argstr system x86_64-linux default.nix` which builds Emacs 30.2 with all 275 tree-sitter grammars included.

### Build Variants

Jotain supports several Emacs build variants:

```bash
# Standard Emacs 30.2 + tree-sitter grammars
nix-build default.nix

# Without tree-sitter grammars
nix-build default.nix --arg withTreeSitterGrammars false

# Terminal-only (no GUI)
nix-build default.nix --arg noGui true

# Pure GTK for Wayland
nix-build default.nix --arg withPgtk true

# Bleeding-edge from git master
nix-build default.nix --arg variant '"git"'

# Incremental GC branch
nix-build default.nix --arg variant '"igc"'

# macOS macport fork
nix-build default.nix --arg variant '"macport"'
```

<Note>
For git/unstable/igc variants, the first build will fail and report the correct hash. Re-run with `--argstr hash "sha256-..."`.
</Note>

## Running

Launch Emacs with the Jotain configuration:

```bash
just run
```

This starts Emacs with `--debug-init` using the Jotain directory as the configuration source.
