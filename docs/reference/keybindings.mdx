## General

| Key | Action |
|-----|--------|
| `C-x j` | Toggle window split direction |
| `C-c u` | Winner undo (restore previous window layout) |
| `C-c r` | Winner redo |
| `C-c t` | Toggle light/dark theme |

## Navigation & search

| Key | Action |
|-----|--------|
| `C-x b` | Switch buffer (consult) |
| `C-s` | Search in buffer (consult-line) |
| `M-s r` | Search across project (consult-ripgrep) |
| `M-s f` | Find file by name (consult-fd) |
| `M-g g` | Go to line |
| `C-x p f` | Project find file |

## Programming

| Key | Action |
|-----|--------|
| `M-.` | Go to definition |
| `M-,` | Go back |
| `M-?` | Find references |
| `C-c l r` | Rename symbol |
| `C-c l a` | Code action |

## Completion

| Key | Action |
|-----|--------|
| `C-.` | Embark act (context action) |

## Git

| Key | Action |
|-----|--------|
| `C-x g` | Magit status |

## Fonts

| Key | Action |
|-----|--------|
| `C-+` | Increase font size |
| `C--` | Decrease font size |
| `C-0` | Reset font size |
| `C-c f i` | Show font configuration info |
