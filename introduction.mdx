# Jotain

**Jotain** is a custom GNU Emacs configuration built from scratch. The name is Finnish for "something."

The project includes:

- A complete Emacs Lisp configuration with modular architecture
- A Nix expression for building Emacs from source with fine-grained control over build options
- Support for multiple Emacs variants (mainline, git, unstable, macport, IGC)
- Built-in tree-sitter grammar support (275 grammars from nixpkgs)

## Why Jotain?

Jotain takes a different approach from framework-based Emacs configurations like Doom or Spacemacs. Instead of layering on top of an opinionated framework, Jotain is built from scratch with:

- **Nix-based builds** — Reproducible Emacs builds with precise control over compile options
- **Modular Elisp** — Clean separation of concerns with dedicated modules
- **Modern Emacs** — Targets Emacs 30+ with native compilation, tree-sitter, and lexical binding throughout
