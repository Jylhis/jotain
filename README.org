#+title: Jotain: A Modern Emacs Configuration

* Introduction

Jotain is a modern, modular Emacs configuration managed with Nix Flakes, emphasizing performance and reproducible development. It provides a complete development environment with LSP support, modern completion (Vertico/Corfu), Git integration (Magit), and sensible defaults.

All Emacs packages are installed via Nix — never via MELPA/ELPA at runtime.

* Features

- *Reproducible:* Managed by a Nix Flake for consistent development environments.
- *Modular:* Configuration is split into logical modules for easy customization.
- *Performant:* =early-init.el= is optimized for fast startup times.
- *Integrated:* Comes with a Home Manager module for easy installation.
- *Comprehensive:* Includes packages for development, writing, Git, AI, and more.
- *Well-tested:* ERT and NMT test suites ensure stability.

* Quick Start

Add Jotain to your =flake.nix=:

#+begin_src nix
{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
    home-manager.url = "github:nix-community/home-manager";
    jotain.url = "github:Jylhis/jotain";
  };

  outputs = { nixpkgs, home-manager, jotain, ... }: {
    homeConfigurations."<your-username>" = home-manager.lib.homeManagerConfiguration {
      pkgs = nixpkgs.legacyPackages.x86_64-linux;
      modules = [
        jotain.homeModules.default
        {
          programs.jotain.enable = true;
        }
      ];
    };
  };
}
#+end_src

** Try without installing

#+begin_src bash
nix run github:Jylhis/jotain
#+end_src

* Configuration

| Option | Default | Description |
|--------+---------+-------------|
| =programs.jotain.enable= | =false= | Enable Jotain |
| =programs.jotain.enableDaemon= | =true= | Run Emacs as systemd/launchd service |
| =programs.jotain.includeRuntimeDeps= | =true= | Install LSP servers, fonts, CLI tools |
| =programs.jotain.extraPackages= | =epkgs: []= | Add extra Emacs packages |

Add extra packages:

#+begin_src nix
programs.jotain.extraPackages = epkgs: [
  epkgs.pdf-tools
  epkgs.org-roam
];
#+end_src

Add personal configuration to =~/.config/emacs/custom.el= (auto-loaded, persists across rebuilds).

* Keybindings

| Key | Action |
|-----+--------|
| =C-x g= | Magit status |
| =C-x p f= | Project find file |
| =C-x b= | Switch buffer (consult) |
| =C-s= | Search in buffer |
| =M-.= | Go to definition |
| =M-,= | Go back |
| =C-c t= | Toggle light/dark theme |
| =C-x j= | Toggle window split direction |
| =C-c u= / =C-c r= | Winner undo/redo window layouts |
