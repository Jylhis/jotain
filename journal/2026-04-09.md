# 2026-04-09

## Issues, questions, and TODO cleanup

### Issues fixed

**Duplicate dired icons** — removed `nerd-icons-dired` use-package block from
`init-ui.el`. Dirvish already renders nerd-icons via `dirvish-attributes
'(nerd-icons ...)` in `init-navigation.el`; the extra hook was causing both to
fire simultaneously, producing two identical icons per entry.

**Relative line numbers** — removed `:custom (display-line-numbers-type 'relative)`
from the `display-line-numbers` block in `init-ui.el`. Line numbers remain active
in `prog-mode`/`conf-mode` but now use the default absolute style.

### Question answered

**package-quickstart** (`questions.md`) — explained what the feature does and why
`package-quickstart-file` is pinned to `var/package-quickstart.el` in
`early-init.el` (must be set before `startup.el` runs `package-activate-all`).
